# Lesson: Linting
## Todo
1. Run Todo App
1. Add Todo and check console log in dev tools
1. Install eslint with npm
1. Run eslint --init
    - Answer question about..
    - ECMAScript 6 features -> yes
    - ES6 Modules -> yes
    - Browser
    - CommonJS -> no
    - JSX -> no
    - Indentation -> spaces
    - Quotes -> single
    - Line Endings -> Unix
    - Semicolons -> yes
    - Format: Javascript
1. Add npm step "lint"

```node_modules/.bin/eslint src/**/*.js```
